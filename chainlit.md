# Multi-Source Research Agent

Powered by **LangGraph** + **Groq (Llama 3.3 70B)** + **DuckDuckGo**

Ask any research question and this agent will:

1. Break it into focused sub-questions
2. Search and scrape real web sources
3. Reconcile conflicting information across sources
4. Write a structured report with citations and confidence levels
5. Reflect and retry if any sub-questions were missed

**No paid search API required.**
